Device

Description

iKnoMors is a Morse code learning device that allows the user to input characters from a USB keyboard that will display the characters typed onto a LCD screen. Once the go button is pressed, the input that is typed on the LCD display will imitate the sound of Morse code through a speaker while also blinking it through a LED for a visual and audible learning experience.

Controls

Keyboard - Plug in keyboard to USB Port for input of characters.
Scroll Up/Scroll Down - Scrolls the screen up and down.
GO - Plays Morse Code through speaker and LED.
CLEAR - Clears screen of all characters.
Volume - Loudness of the speaker
ON/OFF - Turns device on or off.
